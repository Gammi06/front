d1
1) background-color 속성: 배경색만
d2
2) background-image 속성: 이미지를 배경으로 {background-image:url(경로);} -> 기본적으로 width, height가 있어야 정확히 배경이미지가 출력됨 -> w, h에 무한반복되어 이미지 출력
d3
3) background-repeat 속성: 이미지 반복여부 설정 - no-repeat - repeat-x - repeat-y
d4
4) background-position 속성: 이미지 출력 위치 - 기본 (0 0) 왼쪽 상단 - 배경이미지를 반복하지 않고 사용할 것 {background-repeat: no-repeat;}일때 함께 사용
d5
5) background 속성: 1 ~ 4번의 개별속성을 한번에 처리할때 사용 {background: 색상 url(이미지경로) 반복여부 가로위치 세로위치;} * 값을 순서대로 작성하는 것이 중요 * 4가지 속성값을 모두 작성해야하는 것은 아님
d6
6) background-size 속성: 배경이미지 크기 조절 {background-size: 가로크기 세로크기;} -> {background-size: 150px auto;} : 가로크기 150px, 세로크기 auto=자동 (이미지비율에 맞춰서 알아서 인식됨)
d7
7) background-attachment 속성: 배경이미지의 스크롤을 조절함 {background-attachment: fixed;} : 해당태그의 영역에서만 배경이미지가 출력됨 웹브라우저 기준 배경이미지를 고정시킴



css 이미지 스프라이트 기법

  1. 여러개의 이미지를 하나의 이미지로 합쳐서 사용하는 방법 -> 이미지의 관리 용이 / 웹사이트 이미지 로딩시 시간 및 용량을 줄일 수 있음 -> 이미지 적용시 웹표준을 준수해야 하므로 가상대체 텍스트를 사용해야함 (IR: Image Replacement 기법 사용)
