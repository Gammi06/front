스타일 css 속성
: 태그의 모양을 변경하기 위해 style 속성을 사용하게 됨

1. 인라인 스타일 inline style
: 태그에 직접적으로 스타일을 적용하는 방식
 <tag style="property: value"></tag>
 <태그명 style= "스타일속성: 값; 스타일속성: 값;"></태그명>
  -> 세미콜론으로 끊기, 띄어쓰기로 구분함

2. 내부 스타일 internal style
<body>태그에는
<태그명 id="아이디명"></태그명>    : 큰 구조, 영역에 사용(이름 한번만 사용 가능)
<태그명 class="클래스명"></태그명> : 중복디자인, 여러영역에 사용, 중복사용가능

<head>태그사이 <style>태그 작성
태그명{}
#아이디명{}
.클래스명{}

3. 외부 스타일 external style
외부에 css문서를 작성하여 html문서와 연결하여 사용
css문서 첫 줄
@charset "UTF-8";
<link href="경로" rel="stylesheet">
!!! html문서와 css문서는 꼭 연결되어야 함 !!!

* 공간의 값
width 속성: 태그의 가로너비 px
height 속성: 태그의 세로높이 px

- 항상 단위는 px로 작성
- 태그는 사각형(박스모델)으로 표현되는 것이 기본적임
- 배경색이 없어 투명함(공간은 차지하고 있음)
- 공간값을 가질 수 있는 태그는 블록요소만 가능함/인라인요소 불가능
- 태그의 공간값이 적용될 경우, 태그안의 글자내용은 공간에 맞춰
  자동줄바꿈되는 특징이 있음


*테두리
border속성: 태그에 테두리를 추가하는 속성
- 사방으로 기본적용됨
  (border:선두께 선모양 선색상)
  (border:5px solid #f00)  -> 테두리 5두께 기본선 빨간색

- 선모양 대표 3가지: solid / dotted / dashed

한 방향으로 적용하고 싶다면?
border-top      : 위
border-right    : 오른쪽
border-bottom   : 아래
border-left     : 왼쪽

* box-model 박스모델
: 태그가 차지하는 공간을 계산하는 방법
(태그의 기준은 왼쪽상단 꼭지점 0,0 자리 기준)

box-model width: width + padding 좌, 우 + border 좌, 우 + margin 좌, 우
box-model height: height + padding 상, 하 + border 상, 하 + margin 상, 하

* 여백 속성(바깥쪽, 안쪽)

태그의 테두리를 기준으로
margin : 바깥 영역 추가 여백 (배치 이동 안됨, 공간X)
padding: 안쪽 영역 추가 여백 (태그공간이 늘어남, 공간값 계산)

{margin:30px}   : margin, padding 동일하게 사방으로 30px
{padding:30px}

margin-top
margin-right
margin-bottom
margin-left
** padding 값 동일하게 적용됨

margin:값1 값2 값3 값4; : 상 우 하 좌 (시계방향)
margin:값1 값2 값3;     : 상 좌==우 하
margin:값1 값2;         : 상==하  좌==우
margin:값1;             : 기본값 (사방동일)

*태그의 배치를 위한 스타일 속성 (가로배열)

1. float 속성 / ::after 가상 선택자
: 블록요소인 태그를 가로배열 시킬 때 사용
{float:left}  : 왼쪽 -> 오른쪽
{float:right} : 오른쪽 -> 왼쪽

float 적용 시 지켜야할 조건
1) 블록요소 적용(인라인x)
2) 형제관계 성립 태그들만 float 적용이 가능함
   (부모-자식 / 부모-자손 관계태그는 float 적용 못 함)
3) 가로배열되는 형제관계 "모두"에게 float 속성 적용 시켜야 함
4) 부모태그영역의 가로너비값이 1px이라도 모자라면
   한줄에 나란히 되지 않고, 줄바꿈 되는 특징이 있음
   -> 이 특징을 사용해 여러줄 배열 처리함

  float 적용하면 항상 부모태그에게 ::after 가상클래스 선택자를 사용해야 함
  float를 적용시킨 태그는 웹브라우저에서 "뜬다"라는 개념으로 인식함
  -> float가 적용 된 부모태그는 공간값을 인식하지 못함
  -> 높이가 0이므로 자동으로 높이 값을 인식시키기 위해 ::after 사용함

  부모태그::after{content:""; display:block; clear:both; }

  - clearfixed 기능이라고 함
  (해당 기능을 적용하는 방식은 다양함)

  content: "";     <- 가상의 태그를 만들 때 그 내용을 작성하는 content 속성
                     내용이 빈 공간을 만들어주기 위해서

  display: block;  <- 블록요소화 시켜줌
                     (가상 태그는 인라인요소, 그래서 빈공간값을 적용하기 위해서
                      블록요소로 변경해줘야 함)

  clear: both;     <- float: left / right 두개(both) 속성을 무효화

2. display 속성

    1) display:inline-block

    2) table
       display:table
       display:table-cell

    3) flex (Responsive Web)
       display:flex
       display-direction:colmn

    4) css Grid
       display:grid
       grid-template-columns: 100px 100px 100px
       grid-template-row: 100px 100px

////////////////////

{text-indent: 숫자;}
: 첫번째 글자 위치 변경함 (들여쓰기/내어쓰기)
양수(+) > 우측 ||| 음수(-) < 좌측

웹브라우저 좌측으로 이동하는 컨텐츠는 보여주지 않아도 되기 때문에
ui화면에서 사라지거나, 잘리거나 안보이게 됨

이 속성을 활용하여 모든 해상도, 모니터 크기에 무관하게
해당 글자를 안보이게 하는 방법으로
{text-indent: -9999px;}를 사용함
글자만 안보이게 처리할때 주로 사용
