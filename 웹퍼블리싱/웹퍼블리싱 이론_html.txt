* HTML이란?

- 웹브라우저가 알아들을 수 있는 문서의 형식 (html문서)
- html문서는 태그 <tag></tag>로 구성됨
- 태그란? html문서를 구성하는 명령어
- <태그명></태그명>
- 태그 작성시 소문자 사용

* html문서의 기본형식

  <head>

<em></em>: span보다 작은 개념, 전문용어, 번역문구로 활용(인라인요소, 기울임꼴), 웹접근성

* display 속성

1) block요소 (블록요소)
 - 자동줄바꿈, 세로정렬
 - 기본 가로너비값을 전체 100% 인식(부모상속)
 - width, height 공간값 적용 가능
<div> <p> <ui> <li> <ol> <dl> <dt> <dd> <h1~h6>

2) inline요소 (인라인요소)
 - 가로정렬, 줄바꿈X
 - 기본 가로너비값을 자신의 안쪽에 작성된 글자 or 이미지만큼 인식
 - width, height 공간값 적용 안됨
 - 주로 내용(컨텐츠) 담당할 때 사용하는 속성
<a> <span> <strong> <i> <em>

3) {display: inline block}
 - 기본은 inline요소
 - 인라인요소 이면서 block의 (가로, 세로 공간값 인식)특징을 가짐
 - float 대신 대체해서 사용하지 않음
  (가로배열할때 특정영역에서 사용함)
 - <img> <input> ...

4) {display: none}
 - 해당 속성을 가진 태그는 html 구조상에서 삭제되는 형식
 - 공간도 없어지고, ui 화면 출력도 되지 않는 상태
 - 주로 ui user interaction 효과에 사용함

5) {display: block}
 1. 인라인요소인 태그를 block요소화 시켜 공간값을 인식시키는 개념
 2. {display:none}으로 사라진 태그를 {display:block}으로 보이게 할 때 사용

* 기본태그

<a></a>: 링크 역할(인라인 요소)
 - <a>태그 담길 글자, 이미지 클릭하면 원하는 위치로 이동하는 역할
 - href 필수 속성: 링크 이동경로 지정
 - 사용법: <a href="이동경로"></a>
   1) 절대경로: http://~~, C:\user\document\~~
   2) 상대경로: 현재 작성하는 html문서를 기준으로 경로 지정
      같은 파일에 있다면?: 파일명.확장자
      다른 파일에 있다면?: 폴더명/파일명.확장자
	/  : root루트 최상위 폴더
	./ : 현재 위치 기준
	../ : 상위 폴더로

* target속성: 링크에 담긴 문서를 열 장소를 지정함

target= "_blank": 새 탭에서 열기
target= "_self": 기본값, 현재창에서 열기
_parent
_top
<a href="mailto:메일주소@naver.com"></a>
*title= "도움말설명"

////////////////////

<img>: 이미지의 경로를 연결시키는 태그(인라인 요소)
 - src 필수 속성: 이미지 경로 지정
 <img src= "이미지경로">
 - alt 속성: 웹접근성을 위해 이미지 설명글 제공
   (정상출력 경우 글자가 보이지 않음, 이미지가 깨졌을때 화면에 글자가 출력됨)
  <img src="이미지경로", alt="이미지설명글">

* 목록태그 3가지: 모두 블록요소, 세로정렬이 특징

1) <ol> 태그: 순서에 의미O
2) <ul> 태그: 순서에 상관X
3) <dl> 태그: 정의형 목록태그

: ol, ul 모두 <li>태그를 포함
<dl>
  <dt>대표단어</dt>
  <dd>대표단어 설명글</dd>
</dl>
